class pybot.minitel.identification.DeviceSpecs(model, maker, version)[source]¶

Bases: object

Specifications of a given model.

This class is a data holder for identification and capabilities of a model. Instances are created with the content of the device identification data as stored in ROM, which are decoded to provide a more human friendly version.

The decoded information are available in the following attributes :

  • maker
  • version
  • model_specs

See also

ModelSpecs definition to know what is included in the model technical specifications

Parameters:
  • model (str) – model code as stored in ROM
  • maker (str) – maker code as stored in ROM
  • version (str) – version as stored in ROM
